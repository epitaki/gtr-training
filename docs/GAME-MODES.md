# ゲームモード仕様書

## 概要

GTRトレーニングツールには、プレイヤーのスキルレベルに応じた複数のゲームモードが用意されています。

## 実装済みモード

### 1. 初心者用ガイド付きGTR練習

#### 特徴
- GTR初心者向けの学習特化モード
- 右側にリアルタイムガイド表示（900x800px）
- 段階的な学習支援システム
- 配置アドバイザーによるゴースト表示

#### ガイドシステム
1. **最初の2手状態**
   - 2手組み合わせパターン別のおすすめ配置表示
   - AAAA、AAAB、ABAB、ABAC、AABC型それぞれのガイド

2. **基本形作成状態**
   - GTRの必須折り返し部分（1-3列目）の作り方
   - 10AB、11AAB、12BBの配置ガイド

3. **連鎖尾作成状態**
   - Y字形、L字形、階段形の3つの連鎖尾パターン
   - パターン別の配置サンプル表示

4. **GTR完成形作成状態**
   - 検出された連鎖尾タイプに応じた完成形サンプル
   - GTRGuidePatterns.tsから完成形を取得

#### 配置アドバイザー（PlacementAdvisor）
- フィールド状態 + 現在のぷよペアから最適な配置位置を推薦
- **3フェーズ自動検出**:
  - `FOLD_BUILDING`: 折り返し構築中（hasBasicPattern=false）
  - `CHAIN_TAIL`: 折り返し完成、連鎖尾構築中
  - `COMPLETION`: 連鎖尾ぷよ≥6、GTR完成へ
- **5要素スコアリング**: 折り返し進捗、連鎖尾、同色連結、高さペナルティ、連鎖シミュレーション
- **ゴースト表示**: 推薦位置に半透明ぷよ（パルスアニメーション付き）
- **Gキー**でON/OFF切替可能

#### ガイド表示仕様
- **配置**: ゲームフィールド右側
- **サイズ**: フィールドと同じ高さに揃えて表示
- **ぷよ表示**: 実際のぷよスプライト（A→赤、B→緑、C→青、D→黄）
- **背景**: ダーク背景（0x1a1a2e）と紫系枠線（0x4444aa）

#### 学習支援機能
- **↑キー**: 1手巻き戻し機能（GameHistory管理）
- **Pキー**: ポーズ・再開機能
- **Fキー**: 自由落下ON/OFF切替（画面左上に状態表示）
- **Gキー**: 配置アドバイザーON/OFF切替
- **Spaceキー**: 詳細なGTR評価画面表示
- **Escキー**: TOPページに戻る

#### 評価システム
詳細なチェック形式の評価結果（成功✓/失敗✗の両方を表示）：
- ✓/✗ 折り返し（必須パターンの有無）
- ✓/✗ Y字連鎖尾 / L字連鎖尾 / 階段連鎖尾
- ✓/✗ 4連鎖 / 5連鎖
- ✓/✗ 右上が立っている（9行目使用状況）
- ✓/✗ あまりぷよが連結

### 2. GTRスコアアタック

#### 特徴
- 従来の5GTRモードを維持（640x800px）
- 高速ゲームプレイに特化
- インタラクティブな評価システム

#### ゲームプレイ
- 通常のぷよぷよルール
- 2手目組み合わせ制約適用
- ネクスト・ネクネクスト表示（縦型ボックス）
- 壁キック機能

#### 操作
- **Fキー**: 自由落下ON/OFF切替（画面右上に状態表示）
- **Spaceキー**: GTR評価実行
- **評価画面**: Space（続行）/ Esc（TOPへ戻る）選択
- スコア、連鎖数、メッセージ表示

## 共通機能

### 基本操作
- **← →**: 左右移動
- **↓**: 高速落下
- **Z**: 左回転
- **X**: 右回転
- **Esc**: TOPページに戻る

### ビジュアルシステム
- **PuyoRenderer**: 目付きぷよテクスチャ（3段グラデーション + 光沢 + 白目 + 黒瞳）
- **VisualConfig**: 統一されたレイアウト定数
- **アニメーション**: 着地バウンス、連鎖消滅 + パーティクル、スコアフロート

### GTR評価システム
- GTRScoringConfig.tsによる設定値管理
- 必須折り返しパターン検出
- 連鎖尾形状判定（Y字・L字・階段）
- 9行目使用状況評価
- あまりぷよ評価（単色連結で高評価）

### 2手目組み合わせ制約
- AAAA型、AAAB型、ABAB型、ABAC型、AABC型の5種類のみ
- 4色すべてが揃うことはない現実的な制約
- 4色固定（赤・緑・青・黄）

## 技術仕様

### アーキテクチャ
- **React**: TOPページとモード管理
- **Phaser.js**: ゲームエンジン
- **独立シーン**: 各モードは独立したPhaser Scene

### ファイル構成
- `TopMenu.tsx`: メインメニューUI
- `GuidedGame.tsx`: ガイド付きモードのReactラッパー（900px幅）
- `ScoreAttackGame.tsx`: スコアアタックモードのReactラッパー（640px幅）
- `GuidedPracticeScene.ts`: ガイド付きモードのPhaserシーン
- `PuyoGame.ts`: スコアアタック用Phaserシーン
- `PuyoRenderer.ts`: ぷよテクスチャ生成（通常 + ゴースト）
- `VisualConfig.ts`: 統一レイアウト・カラー定数
- `PlacementAdvisor.ts`: 最適配置推薦アルゴリズム
- `PlacementAdvisorConfig.ts`: 推薦スコアリング定数

### 状態管理
- `GuideManager.ts`: ガイド状態の遷移管理
- `GameHistory.ts`: 巻き戻し用のゲーム履歴管理
- `GTRGuidePatterns.ts`: ガイド表示用のパターンサンプル
